# CROPDX — Crop Disease Diagnosis Expert System

An AI-based Rule-Based Expert System developed using Turbo Prolog 2.0 for diagnosing crop diseases.

## Features
- Offline disease diagnosis
- Rule-based inference engine
- Backward chaining
- 4 supported crops
- 8 disease detections
- Interactive CLI system

## Technologies Used
- Turbo Prolog 2.0
- DOSBox 0.74
- Rule-Based AI
- Expert Systems

## Supported Crops
- Rice
- Potato
- Tomato
- Wheat

## Project Structure

/source_code → Prolog source code  
/screenshots → DOSBox execution screenshots  
/poster → Academic poster PDF  
/report → Documentation/report  

## Execution Environment

This project was developed and executed using:
- Turbo Prolog 2.0
- DOSBox
- Windows 10

Note:
The source code may not run directly in modern Prolog environments because Turbo Prolog 2.0 uses older syntax.
